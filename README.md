# Simple Wallet
A simple crypto wallets service. You can create or add exists crypto wallets. Management via TG or API.

### Stack
Nest.js, Kafka, PostgreSQL, Sequelize, Redis, Jest

### Build
```shell
npm run build
```

### Run
```shell
npm run start
```

### Run in Docker
```shell
npm run containers:up
```

### Apply migrations
```shell
npm run migrations:up
```
